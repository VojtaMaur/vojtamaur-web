## MANIFEST

This website is a static project focused on long-term content preservation.

All content is stored as files without reliance on databases or external systems. The final website is generated as a static output that can be easily copied, archived, and reconstructed.

The project is distributed across multiple independent systems (repositories, archival services, snapshots) to minimize the risk of data loss.

The goal is to ensure that the content can be reconstructed even if parts of the infrastructure cease to exist.
